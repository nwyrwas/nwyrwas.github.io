About This Project

This capstone represents the culmination of my Computer Science degree at Southern New Hampshire University, demonstrating my ability to design, secure, and deploy full-stack applications. The Animal Shelter Data Analytics Dashboard integrates Python Dash, MongoDB, and Matplotlib to deliver an interactive, data-driven web experience. It provides real-time insights into animal intake and adoption trends through secure queries, visualizations, and map-based data representation.

The project expands upon my original CS-340 Client-Server Development artifact by enhancing usability, scalability, and data visualization. It incorporates key software engineering principles, ensuring maintainability, security, and performance through modular design and adherence to SEI CERT and OWASP standards.

Technical Overview

  • Frontend: Dash Components (HTML/CSS/JavaScript rendered in Python)
  • Backend: Python Flask + Dash callbacks for dynamic UI and database control
  • Database: MongoDB using parameterized queries and optimized indexing
  • Visualization: Matplotlib, Plotly, and Leaflet map integration
  • Environment: Virtualenv, macOS, VS Code, GitHub

Key Features

  • Interactive search and filtering of animal shelter records
  • Dynamic charts showing breed and outcome trends
  • Real-time geographic mapping of animal outcomes
  • Secure CRUD and database access with validated input
  • Modern dark-mode UI for accessibility and professional display

Setup & Execution

To run the dashboard locally:

git clone https://github.com/nwyrwas/CS-499-Capstone-Project.git
cd CS-499-Capstone-Project/Final_Project
python3 -m venv dashenv
source dashenv/bin/activate   # On macOS/Linux
dashenv\Scripts\activate      # On Windows
pip install -r requirements.txt
python app.py

Once the server starts, open your browser and navigate to: http://127.0.0.1:8060

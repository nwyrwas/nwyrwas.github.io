Original Artifact – CS-340 Client-Server Development

The original artifact was developed in the CS-340 Client-Server Development course. It consisted of a Python Flask web application integrated with MongoDB to manage an animal shelter’s database. The system implemented full CRUD functionality through a class called AnimalShelter, which allowed authenticated users to perform create, read, update, and delete operations.

This foundational artifact demonstrated my early understanding of API endpoints, data persistence, and backend interaction.

Enhanced Artifact – CS-499 Capstone Project Dashboard

For my capstone enhancement, I transformed the original server-based application into a dynamic full-stack data analytics dashboard using Python Dash, MongoDB, Matplotlib, and Leaflet. The upgraded system enables users to visualize adoption and rescue data in real-time with advanced search, filters, and mapping capabilities. It also implements dark-mode UI styling, performance-optimized queries, and chart rendering that displays interactive adoption insights.

This enhancement demonstrates practical skills in UI/UX design, data visualization, and distributed system integration while following professional software development standards such as modular architecture, code reuse, and secure database access.
